project

JobGlass

A job tracker web application built using React and Vite, designed to streamline and enhance the job application process. With intuitive user interfaces and powerful features, our app empowers users to manage their job search efficiently and effectively.

Alumni Reunion

Alumni Reunion app is a dynamic platform designed to bring graduates together, powered by Dart technology. With its robust features and user-friendly interface, the app facilitates seamless networking, event planning, and communication among alumni.

Lexical Analyzer

Lexical Analyzer is a robust tool meticulously crafted to streamline the process of parsing and analyzing code, ensuring accuracy and efficiency in software development workflows. By breaking down source code into its fundamental lexical components, our analyzer provides developers with invaluable insights and optimizations for their projects.
